# Overview

This is backend application set up using Node.js, Express.js, Mongoose, and MongoDB. hunder client was used in testing the endpoints/routing.

# Prerequisites

Node.js and npm (Node Package Manager) was installed on my computer.

MongoDB and Express was installed and running on my computer.

Thunder Client is a command-line tool for testing API endpoints.was installed as an extention in my vscode.

# Setting up the project

Created a new directory for my project and was navigated to the command line
initialized a new Node.js project by running npm init

Installed the following dependencies by running the following command: npm install express mongoose

Created a new file called app.js in the root of your project directory. This will be the entry point for your application.
